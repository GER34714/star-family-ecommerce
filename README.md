# Star Family E-Commerce

Aplicación de e-commerce para Star Family, un negocio mayorista de alimentos.

## Características

- 🛒 Catálogo de productos con filtrado por categorías
- 📱 Carrito de compras funcional
- 💬 Pedidos vía WhatsApp
- 🔐 Panel de administrador
- 🎨 UI moderna con animaciones
- 📦 Base de datos en Supabase

## Configuración

### Antes de usar:
1. **Número de WhatsApp**: Cambia `WHATSAPP_NUMBER` en la configuración
2. **Supabase**: Configura tu URL y clave de Supabase
3. **Contraseña admin**: Cambia `ADMIN_PASS` para mayor seguridad

### Instalación

```bash
cd "star family"
npm install
npm start
```

O abre directamente `index.html` en tu navegador para una versión standalone.

## Estructura del Proyecto

- `StarFamilyApp.jsx` - Componente principal de React
- `index.html` - Página HTML para standalone
- `package.json` - Dependencias del proyecto
